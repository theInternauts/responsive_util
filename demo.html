Available console commands

  • $.fn.responsive_util.hide() — Hides the context indicator
  • $.fn.responsive_util.show() — Shows the context indicator
  • $.fn.responsive_util.destroy() — Destroys the context indicator and disables the plugin
    This can be triggered by double-clicking the context indicator, as well.

Configurable Options

  • timer: integer - millesconds before the $.fn.responsive_util.hide() function is called to auto-hide the context indicator
    Example: { timer: 1000 } = one second delay
